fleurish

The very first full-stack application I worked on. Fleurish is an event mapper where users were able to search for and post events that were occurring in the city. Created events were sent to a database where the information was then sent back to the app to reflect the user's input. This is a single-page application built using Angular for the frontend and Node.js for the backend. You can check it out here
